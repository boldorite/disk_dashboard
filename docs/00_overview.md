# Overview

## Purpose
연세대 미래캠 직원용 폴더관리 프로그램. 사용자가 관리할 폴더를 등록하면 그 폴더와 이하 폴더의 폴더트리, 파일 갯수, 전체 파일 사이즈를 대시보드로 보여준다.

## Current Status
- FastAPI 백엔드 + 순수 HTML/CSS/JS 프론트 구현, 로컬 실행 확인 (2026-07-08).
- 폴더 등록/제거/갱신/완전초기화, 폴더트리, 막대바, 탐색기 열기 동작.
- CHO Wiki 표준 구조(.llmwiki, docs/) 편입.

## Core Value
- 관리 폴더별 폴더트리와 재귀 파일수/용량을 한눈에 확인.
- 막대바로 디스크 점유율 또는 상위폴더 대비 비율 시각화.
- 다중 선택 폴더 등록(체크박스 · Ctrl · Shift), 폴더 바로 열기.

## Constraints
- 토큰/비밀번호/개인정보/내부계정은 어떤 파일에도 넣지 않는다.
- `config.json`(로컬 관리 폴더 경로)은 Git 원격에 올리지 않는다.
- Windows 전용 기능: 폴더 열기(os.startfile/explorer), 드라이브 브라우저.
- 포트는 5000~6000 범위에서 자동 선택.
